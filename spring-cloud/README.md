# Spring Cloud

### CI/CD
- 지속적인 통합
    - Jenkins, Team CI, TravisCI
    - 통합 서버, 소스관리, 빌드도구, 테스트 도구
- 지속적인 배포
    - Continuous Delivery (수동 반영)
    - Continuous Deployment (자동 반영)
- 카나리 배포와 블루그린 배포
    - 카나리 배포
        - 95% 기존버전 사용자
        - 5% 새 버전 사용자
    - 블루그린 배포
        - 100% 사용자가 이전 서비스 -> 새 서비스

### DevOps
